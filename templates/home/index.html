Welcome!

I created this website to visuaize the data from a 1KB (1024 bytes) Arduino Uno EEPROM. As a first step I will implement images with only two colors. Each byte can store the values of 8 pixels, that means 8*1024=8192 pixels which we can display in 64 rows and 128 columns. Later I will probably add multicolor (around 4) images but then I will need to reduce the number of pixels.

Arduino UNO

The Arduino Uno is an open-source microcontroller board based on the Microchip ATmega328P microcontroller (MCU) and developed by Arduino.cc and initially released in 2010.
